# Macoder 서버 (패션 컨설팅 플랫폼)  
사용자와 스타일리스트를 매칭하여 패션 컨설팅을 중계하는 플랫폼 서버 

주요 기능:
 - 회원 CRUD
 - 로그인 인증, 인가 
 - 회원 -> 스타일리스트 등업
 - 컨설팅 신청, 답변
